I'm **Jyothula Bhagyasri**, a passionate **Data Specialist** with 1+ years of experience transforming raw data into meaningful insights. I love building data pipelines, interactive dashboards, and low-code automation solutions that help businesses make smarter decisions.

- 🔍 Currently working with **Power BI, Snowflake, Dataiku & Power Platform**
